# Agents

This directory defines AI agents that can be invoked for specific project
workflows. Each agent file specifies capabilities, permissions, provider
preferences, and behavioral instructions.

## Available Agents

- **code-reviewer.md** -- Reviews code for quality, security, and convention
  adherence. Invoke when reviewing PRs or before merging.
