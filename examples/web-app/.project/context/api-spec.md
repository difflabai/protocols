The OpenAPI spec defines all public REST endpoints for the TaskFlow API.
It is the source of truth for request/response schemas, authentication
requirements, and pagination contracts.

The spec is maintained in the repository root as `openapi.yaml` and is
published to the URL above on every merge to main.
